# AI Development Environment Setup

## Overview

This repository documents the setup process for an AI-assisted development environment using modern development tools including Cursor IDE, Claude Code, Codex, Git, and GitHub.

The objective of this project was to configure the required tools, establish a GitHub workflow, and document the setup process in a structured and professional manner.

This repository also serves as a portfolio artifact demonstrating technical setup capability, documentation practices, independent research, and familiarity with AI-assisted development workflows.

* Installed Cursor IDE successfully
* Verified installation and functionality

### Result

## Step 2 — Installing Claude Code Extension

Installed the Claude Code extension from the Cursor Extensions Marketplace.

### Actions Completed

* Opened the Extensions panel inside Cursor
* Searched for "Claude Code"
* Installed the extension
* Logged into the extension successfully

### Result

Claude Code integration was successfully configured.
