# Research Project

## Topic
YouTube Content Strategy for B2B SaaS

---

## Objective

This research project analyzes how successful B2B SaaS founders, marketers, and creators use YouTube to build audience trust, generate demand, educate prospects, and drive business growth.

The goal is to identify repeatable content frameworks, distribution strategies, and audience-building techniques that can later be turned into a practical B2B SaaS YouTube playbook.

---

## Data Collection & Automation Workflow

To make the research process faster and more scalable, I implemented an automation workflow using **n8n**.

- YouTube video URLs are sent into an n8n automation workflow
- The system automatically transcribes the video content
- Transcripts are then used as the raw input for content analysis and note creation
- This removes the need for manual transcription and speeds up data collection significantly

This allows the research process to focus more on **analysis and synthesis**, rather than manual extraction of content.

---

## Repository Enhancements

To improve clarity and documentation of the workflow, the repository now includes:

- Screenshots of the n8n automation workflow
- Visual breakdown of the transcription pipeline
- Supporting context for how YouTube content is processed into structured notes

These additions make the system easier to understand, replicate, and extend for future use cases.

---

## Outcome

This setup enables:

- Faster ingestion of YouTube content into structured research notes
- Consistent formatting across all analyzed content
- A scalable foundation for building a B2B SaaS YouTube strategy knowledge base
